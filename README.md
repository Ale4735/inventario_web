# Sistema Web de Gestión de Inventario

## Descripción
Aplicación web para la gestión de inventario que permite registrar, consultar,
actualizar y eliminar productos, con autenticación de usuarios y control de stock.

## Tecnologías utilizadas
- Node.js
- Express
- MySQL
- MySQL Workbench
- HTML5
- CSS3
- Bootstrap
- JavaScript

## Estructura del proyecto
- backend/: lógica del servidor
- frontend/: interfaz de usuario
- db/: scripts de base de datos

## Instrucciones de ejecución
1. Clonar el repositorio.
2. Importar la base de datos desde el archivo inventario.sql.
3. Ejecutar el backend con Node.js.
4. Abrir el frontend en el navegador.
